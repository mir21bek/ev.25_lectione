git =распределение система контрооля версий это система для отслеживание  и ведение истории изменение файлов, в вашем проекте 
репозоторий - хранилище вашего кода и истории его изменений  
github- веб сайт для размещение git-репозоториев в совместнрой разработке проектов 

команды gitgit connit 
git init -она создает новый локанфый гит репозоториев у вас на пк, в той директории в которой была прописана команда 

git add-команда для добавления ФАЙЛОВ В СИСТЕМУ ОТСЛЕЖИВАНИЕ гита , после это команда изменение начинают отслеживаться гитом 

git commit -m 'comment' - как только достигаемт определенного момента в разработке ? то тогда мы сохраняем и комментируем изменение в локалный репезитор 

git remote add - это команда для того чтобы связать наш локалный репезиторий с удаленным репезиторием , то ксть добавляет подключение к репезиторий  на гитхабе. git remote add <name:origin> ссылка_на_репозиторий:ssh>
 git push -команда для отправки всех ваших изменений на локальном репозитории в удаленный репезиторий git push origin main
